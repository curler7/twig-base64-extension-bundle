# Curler7 Twig Base64 Extension Bundle
Convert image to base64 string

## Installation
```console
composer require curler7/twig-base64-extension-bundle
```

## Usage
```twig
<img src="{{ 'img/logo.svg'|image64 }}" alt="Logo" />
```

## Running Tests
To run tests, run the following command
```sh
docker compose up -d
```
